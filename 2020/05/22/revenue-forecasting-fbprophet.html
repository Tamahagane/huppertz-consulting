import pandas as pd
from fbprophet import Prophet
import matplotlib.pyplot as plt

pd.set_option("display.precision", 3)

The dataset includes data from 2012 to October 2017. We will choose a '''testing_cutoff_data''' before which we will take the data into consideration for training our model and after which we will use the data for evaluating the accuracy of our model.

testing_cutoff_date = "2016"  # only use training data before this year, then compare with data from this year on
sampling_frequency = "W"  # we can calculate everything based upon daily, weekly or monthly data

df = pd.read_csv("daily_revenue.csv", sep=",", decimal="-", encoding="utf-8")
df['ds'] = pd.to_datetime(df['ds'])

df_train = df[df['ds'] < testing_cutoff_date] # split dataset into train and test timeframe
df_test = df[df['ds'] >= testing_cutoff_date]

df_train_resampled =df_train.set_index('ds').resample('1'+sampling_frequency).sum()['y']
df_train_resampled = df_train_resampled.reset_index()

Training

fbprophet can take holidays into consideration and will fit these as special dates. Later on, we can then evaluate, how these holidays affect the sales. Given that we are analyzing a German online shop for need-to-have materials in a mostly professional environment, we will likely see an almost 100% reduction in sales on the holidays.

m = Prophet()
m.add_country_holidays(country_name='DE')
m.fit(df_train_resampled)

if sampling_frequency == 'D':
    future = m.make_future_dataframe(periods=730, freq=sampling_frequency)
elif sampling_frequency == 'W':
    future = m.make_future_dataframe(periods=104, freq=sampling_frequency)
elif sampling_frequency == 'M':
    future = m.make_future_dataframe(periods=24, freq=sampling_frequency)
else:
    print("No valid samplig frequency given!")

forecast = m.predict(future)
